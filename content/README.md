# 内容编辑指南

这个目录包含全域救援网站的全部内容。你只需要编辑 `.md` 文件，然后运行构建命令即可。

## 快速上手

1. 找到你要编辑的文件（按内容类型分目录）
2. 用任意文本编辑器打开 `.md` 文件
3. 修改内容，保存
4. 运行 `npm run build:content` 生成数据
5. 运行 `npm run dev` 在本地预览

## 目录结构

```
content/
├── missions/        # 行动记录（每场任务一个文件）
├── people/          # 队员档案（每人一个文件）
├── capabilities/    # 现场能力（每种能力一个文件）
└── journal/         # 战地记录（每篇文章一个文件）
```

## 文件格式

每个 `.md` 文件有两部分：

**第一部分：`---` 包裹的元数据（Frontmatter）**
这是结构化数据，格式为 `键: 值`。必填字段在模板中已标注。

**第二部分：正文**
用 `## 标题` 分段，对应网页详情页的各个区块。段落之间空一行。

## 如何新增内容

复制对应目录下的 `_TEMPLATE.md`，改名为新编号（如 `059.md`），填写内容即可。

## 图片

### 本地图片（推荐）

把图片放到 `static/images/content/` 目录下，然后在 Markdown 里用标准语法引用：

```
![图片说明](/images/content/文件名.jpg)
```

系统会自动把图片插入到它所在的位置。支持 jpg、png、webp、gif 格式。

### Unsplash 主题图片

也可以使用预设的在线主题图片，指定 `after` 控制跟在哪个段落之后（段落编号从 0 开始）：

```
![图片说明](after:0, theme:mountain)
```

可选主题：
- `mountain` — 山岳
- `forest` — 森林
- `rain` — 雨天
- `night` — 夜间
- `terrain` — 地形
- `portrait` — 人像
